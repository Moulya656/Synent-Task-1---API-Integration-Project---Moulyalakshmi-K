# Weather Dashboard - API Integration Project

This is a responsive Weather Dashboard developed as part of my Web Development & Designing Internship at Synent Technologies. The application fetches live weather data using the Open-Meteo API and displays weather information based on the city searched by the user.

## Weather Features

- Search weather details by city name
- Auto city suggestions while typing
- Fetch live weather data using API integration
- Display current temperature
- Display humidity information
- Display wind speed
- Dynamic weather data display
- Error handling for invalid city searches
- Responsive and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
- Open-Meteo API

## How It Works

Users can search for a city using the search bar. The application fetches real-time weather data from the Open-Meteo API and dynamically displays the temperature, humidity, and wind speed. City suggestions are provided while typing to make searching easier.

## API Integration

The project uses the Open-Meteo API to retrieve weather information. JavaScript Fetch API is used to request and process the API data and dynamically update the user interface.

## Project Purpose

The purpose of this project is to understand API integration, asynchronous JavaScript, Fetch API, and dynamic DOM updates while developing a functional weather application.
